How to Run Any Project:

Step 1:
Clone the repository.
git clone https://github.com/KallepalliShivaKumar/SpringBoot.git

Step 2:
Open the project folder you want to run using Spring Tool Suite.

Step 3:
Right-click on the project and choose Run As -> Spring Boot App.
