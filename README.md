# Spring Boot Practice Workspace

This repository contains my daily practice projects created using Spring Tool Suite (STS).
All projects are created using Spring Starter Project and are intended purely for learning
and practice purposes.

## Purpose

-- Practice Spring Core concepts  
-- Understand Dependency Injection (Setter, Constructor, XML, Annotations)  
-- Learn Spring Boot fundamentals  
-- Improve Java backend development skills  
-- Track daily learning and progress using GitHub

## Project Structure

## 01-spring-core

-- Basic Spring Core examples
-- Bean configuration
-- Simple Java + Spring integration programs

## 02-dependency-injection

-- Setter Injection examples
-- Constructor Injection examples
-- XML based configuration
-- @Autowired and @Qualifier practice programs

## 03-spring-boot-basics

-- Simple Spring Boot applications
-- Starter projects created using Spring Initializr
-- Configuration using application.properties

## Technologies Used

-- Java
-- Spring Core
-- Spring Boot
-- Maven
-- Git and GitHub

## How to Run Any Project

Step 1:
Clone the repository

git clone https://github.com/KallepalliShivaKumar/SpringBoot.git

Step 2:
Open Spring Tool Suite (STS)

Step 3:
Import the project
-- File -> Import -> Existing Maven Projects
-- Select the required project folder
-- Finish

Step 4:
Run the application
-- Right click on the project
-- Run As -> Spring Boot App / Java Application
