**Spring Tools Practice Workspace**
-----------------------------------

This repository contains my daily practice projects created using Spring Tool Suite.
All projects are created using Spring Starter Project and are used to practice Java
and Spring concepts.

Purpose of this Repository

The purpose of this repository is:
- To practice Spring Core and Spring Boot concepts
- To understand dependency injection concepts
- To improve Java backend development skills
- To track daily learning using GitHub

Technologies Used

- Java
- Spring Boot (Spring Starter Project)
- Spring Core
- Git and GitHub

Projects Included

Each folder in this workspace is an independent practice project.

These projects are created only for learning purposes.

How to Run Any Project

Step 1:
Clone the repository.
git clone https://github.com/KallepalliShivaKumar/SpringBoot.git

Step 2:
Open the project folder you want to run using Spring Tool Suite.

Step 3:
Right-click on the project and choose Run As -> Spring Boot App.

- How Spring manages objects
- How dependency injection works
- How to structure Spring applications
